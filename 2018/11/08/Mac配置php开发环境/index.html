Mac配置php开发环境

Posted on 2018-11-08 | | 浏览 次
Words count in article: | Reading time ≈

一 Apache配置

  • MACOS系统是自带Apache的，所以只需要启动对应的服务就行了。启动Apache的命令是Sudo apachectl start,接下来就可以查看版本 httpd -v

  • 常用的apache的命令

    • 重启apache sudo apachectl restart
    • 关闭apache sudo apachectl stop

  • 启动apache后，到浏览器地址栏中输入localhost，如果出现“It Works”,说明apache已经成功启动

二 php配置

  • MAC系统也是自带php的，只需要在apache的配置文件中田间apache对php的支持就行了。

  • 编辑apache配置文件httpd.conf,用vim打开 sudo vim /etc/apache2/httpd2.conf,然后输入电脑密码就能直接进入配置文件，找到“LoadModule php5_module libexec/apache2/libphp5.so”，按‘i’进入编辑模式，将注释“#”去掉，然后按‘esc’退出编辑模式，然后再按“:wq”回车，保存并退出

  • 终端输入 php -v 查看php版本

  • 配置完成后重启apache，sudo apachectl restart，然后在终端输入命令“open /Library/WebServer/Documents”，该目录即为apache的目录，在此目录下新建一个“info.php”的文件，文件内容为<?php phpinfo(); ?>，然后在浏览器中输入localhost/info,就能看到php的相关信息

二 php配置

  • 下载对应的dmg镜像文件 [http://dev.mysql.com/downloads/mysql/] (http://dev.mysql.com/downloads/mysql/)

  • 安装mysql，安装完成会弹出一个有root的临时密码，记下这个密码。

  • 配置路径

    • 终端输入pico .bash_profile，然后回车
    • export PATH=$PATH:/usr/local/mysql/bin/，将这一段加在最后一行，然后“:wq”保存退出
