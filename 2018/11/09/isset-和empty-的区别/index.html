isset()和empty()的区别

Posted on 2018-11-09 | | 浏览 次
Words count in article: | Reading time ≈

isset()

  • 一般用来检查变了是否被赋值

  • 格式：bool isset ( mixed var [, mixed var [, …]] )

  • 返回值

    • 若变量不存在则返回false
    • 若变量存在且为null，则返回false
    • 若变量存在且不为null，则返回true
    • 同时检查多个变量时，每个单项都符合上一条时才返回true，否则结果false

说明

  • 使用unset()释放变量后，将不再是isset();
  • isset()只能用于变量，传递任何其他参数将造成其解析错误
  • 检测常量是否已赋值可食用defined()

empty()

  • 判断值是否为空

  • 格式 bool empty ( mixed var )

  • 返回值

    • 若变量不存在则返回true
    • 若变量存在且其值为””,0,”0”,null,false,array(),var $var,以及没有任何属性的对象，则返回true
    • 若变量存在且其值不为””,0,”0”,null,false,array(),var $var,以及没有任何属性的对象，则返回false

说明

  • empty()的返回值=!(boolean) var,但不会因为变量为定义而产生警告信息
  • empty()只能用于变量，传递任何其他参数都会造成parse error
